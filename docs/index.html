📦 Installation

Prerequisites

  • Python 3.7 or higher
  • pip package manager
  • Internet connection for API calls

Quick Install

$ git clone https://github.com/yourusername/aserecon.git
$ cd aserecon
$ pip install -r requirements.txt
$ python asetool.py -h

🔌 Shodan Integration

ASERecon optionally integrates with Shodan for enhanced reconnaissance capabilities.

Getting a Free API Key:

1. Visit https://account.shodan.io/register
2. Create a free account
3. Navigate to your API key section
4. Copy your API key

What Shodan Adds:

  • 🔌 Open port enumeration
  • 🛡️ Service fingerprinting
  • 📊 Known vulnerability detection (CVEs)
  • 🌍 Geographic and ISP information
  • 📜 Historical data
Note: Free Shodan API tier has rate limits (1 query/second). The tool works perfectly without Shodan using API-free OSINT sources.

❓ Frequently Asked Questions

🔹 Do I need a Shodan API key?

No. The tool works perfectly without Shodan using API-free OSINT sources (DNS, certificate transparency, WHOIS, etc.). Shodan only adds optional port scanning and vulnerability detection.

🔹 Is this tool free?

Yes, completely free and open-source. No hidden costs or premium features.

🔹 Can I scan any website?

NO! Only systems you own or have written permission to test. Unauthorized scanning is illegal.

🔹 How long does a scan take?

Typically 30-60 seconds depending on network speed, target response time, and enabled features.

🔹 What platforms are supported?

Linux, macOS, and Windows (with Python installed). Any platform that supports Python 3.7+.
